# Civil Engineering Web App

A restructured full-stack web application for civil engineering quantity surveying and reinforced concrete calculations.

## Project structure

- `src/backend/`
  - `server.js`: application entrypoint
  - `routes/`: route modules (`pages`, `download`)
  - `config/`: backend constants
- `public/pages/`: HTML pages
- `public/assets/js/`: frontend scripts
- `public/assets/css/`: frontend stylesheets

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`.
