- Excel Verification
- Unit Tests

Internal Validation

# Silver Certification

Requirements:

- Textbook Examples
- Academic References

Status:

Professional Validation

# Gold Certification

Requirements:

- ETABS Comparison
- STAAD Comparison
- PCA Comparison
- NSCP Examples

Status:

Commercial Release Ready
