Foundation Design

What format? (one row = one foundation)
The uploaded file must contain a sheet named DESIGN PARAMETERS. Each row is one foundation, so you can design any number of footings in a single upload.
  • Row 1 — parameter headers. Add an optional Label column at the start to name each footing (otherwise they're auto-numbered).
  • Rows 2+ — one row per foundation. Cells map by header name; unknown headers are ignored and listed in the result banner.
  • Sample headers: Structure Type, Allowable Load P (kN), f'c (MPa), Bar Diameter (mm), … (40 supported labels — see the PARAMETER GUIDE sheet in the template for the full list and allowed values).
Hit the Download blank template link above for a ready-to-fill copy with three sample footings (one of each structure type) and a guide sheet. After upload, every footing runs through the full design pipeline and a consolidated schedule is shown at the bottom.

Footing Layout

If one edge is a property line and the other is free, a Rectangular footing is used. If both edges are on property lines, a Trapezoidal footing is used. The overhang is the slab projection past the column face toward the property line; a free edge has none.
