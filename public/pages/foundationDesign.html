What format?
The uploaded file must contain a sheet named DESIGN PARAMETERS with two columns:
  • Column A — parameter label (e.g. Allowable Load P (kN))
  • Column B — numeric value or text option
Row 1 is treated as a header. Unknown labels are ignored. Hit the Download blank template link above to get a ready-to-fill copy with every supported label and a Notes column explaining allowed values.
